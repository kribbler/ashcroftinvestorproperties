== Description ==
[PhpSword Google Analytics WordPress plugin](http://69plugins.com/free-plugins/phpsword-google-analytics/ "PhpSword Google Analytics WordPress plugin Link at 69plugins.com") lets you add Google Analytics code easily to your WordPress website. This is one of the simplest and easy to use WordPress plugin to insert Google Analytics or any other analytics code easily to your Website built using WordPress. The following website analytics codes can be installed by using this plugin.

* Google Analytics
* Yahoo Web Analytics
* Clicky
* Piwik
* Mint
* Church Analytics
* KISSmetrics
* Open Web Analytics
* Clicktale
* CrazyEgg
* CloudFlare

= PhpSword Google Analytics WordPress Plugin Features =
* Very simple and easy to use.
* No complex menus or setting.
* Just copy and past your Google Analytics code to the given box and done.
* Enable or disable the tracking code on the website.

== Installation ==
= Search and Install from WordPress admin panel =
1. If you are logged in inside your WordPress admin panel, go to Plugins menu and click "Add New" button.
2. On Install Plugins page, under Search navigation menu, enter "PhpSword Google Analytics" in the search bar and click search plugins.
3. You will find PhpSword Google Analytics plugin By Pradnyankur Nikam in the list.
4. Click "Install Now" button to install the plugin or click "Details" to read more. Install now link is also available on details popup window.
5. Once installed you need to activate the plugin to use it on your website.

= Manual Installation =
1. Click "Download Version 1.0" button on plugin info page to download and save the plugin on your computer.
2. You will get "phpsword-google-analytics.zip" file.
3. Login to your WordPress admin panel, go to Plugins -> Add New -> select "Upload" navigation on the page. Click the browse button, select downloaded "phpsword-google-analytics.zip" file from your computer and click Install Now button. Go to plugins and activate the PhpSword Google Analytics plugin from the list. OR Extract "phpsword-google-analytics.zip" file and upload all the files inside "phpsword-google-analytics" folder to your website's "wp-content/plugins" directory. Now login to your WordPress admin panel go to plugins and activate the PhpSword Google Analytics plugin from the list.
4. Reload WordPress Admin Panel & you will get a new menu "PhpSword Analytics" on left sidebar.
5. Click and open "PhpSword Analytics" page. Here you can insert and save your Google Analytics code or other website's analytics code.
6. You can also enable or disable the analytics code by selecting respected dropdown menu.

= How to turn off the analytics? =
On "PhpSword_Analytics" page you can find dropdown menu to enable or disable the plugin. Just choose appropriate option and save.
